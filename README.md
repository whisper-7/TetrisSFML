## 📋 项目介绍
这是一个使用 **C++** 和 **SFML** 图形库实现的经典俄罗斯方块游戏。项目从零开始，逐步实现了完整的游戏逻辑和界面。

## ✨ 已完成功能

### 🎮 核心玩法
- ✅ 左右移动方块
- ✅ 方块旋转（4种方向）
- ✅ 自动下落
- ✅ 方块固定
- ✅ 碰撞检测（边界、方块重叠）
- ✅ 消除满行
- ✅ 空格键快速落到底
- ✅ 游戏结束检测

### 📊 界面功能
- ✅ 灰色游戏区域 + 黑色方块
- ✅ 网格线显示
- ✅ 下一个方块预览
- ✅ 实时分数显示
- ✅ 游戏结束提示
- ✅ 重新开始（R键）

### 🏗️ 代码结构
- ✅ 常量统一管理
- ✅ UI类封装所有绘制
- ✅ 函数职责单一
- ✅ GitHub 版本管理

## 🎮 操作方法

| 按键 | 功能 |
|------|------|
| ← → | 左右移动 |
| ↑ | 旋转方块 |
| ↓ | 加速下落 |
| 空格 | 直接落到底 |
| R | 重新开始游戏 |

## 🛠️ 开发环境

- **操作系统**：Windows 10/11
- **开发工具**：Visual Studio 2022
- **图形库**：SFML 2.6.1
- **版本管理**：Git + GitHub
